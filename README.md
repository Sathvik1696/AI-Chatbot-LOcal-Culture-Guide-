📌 AI Local Culture Guide Chatbot

An interactive AI-powered chatbot designed to provide local cultural guidance, traditions, and etiquette for different places in India. This project helps users understand cultural norms before visiting religious or heritage locations.

🚀 Features
💬 Interactive chat interface (HTML-based UI)
🏛️ Cultural guidance for temples and local places
👕 Dress code recommendations
🙏 Religious etiquette and behavioral rules
📱 Simple and responsive UI design
⚡ Lightweight — runs directly in the browser
🧠 Project Overview

This chatbot focuses on context-aware cultural assistance, similar to emerging AI systems that provide localized knowledge and culturally relevant responses. Such systems are increasingly used to bridge information gaps and improve accessibility to regional knowledge.

The chatbot responds to queries like:

"What to wear in Tirumala?"
"Rules in Golden Temple?"
"Temple etiquette in India"
