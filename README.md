# AI Local Culture Guide 🌍🤖

An AI-powered chatbot that helps travelers and users understand **local culture, traditions, dress codes, temple rules, etiquette, and regional customs** across different places in India.

Built with a clean conversational UI inspired by modern messaging platforms, this project aims to provide culturally aware guidance in a simple and accessible way.

# ⚙️ Installation

Clone the repository:

```bash

```

Open the project folder:

```bash
cd AI-Chatbot-LOcal-Culture-Guide-
```

Run the HTML file directly:

```bash

```

Or use VS Code Live Server extension.
